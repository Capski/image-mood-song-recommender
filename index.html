Image Mood Song Recommender

This application analyzes uploaded images to detect dominant colors and mood, then recommends songs from Spotify that match the detected mood.

Ready to try it out? Deploy your own instance using the code below!

View on GitHub

Getting Started

Clone the repository: git clone https://github.com/Capski/image-mood-song-recommender.git
Install dependencies: pip install -r requirements.txt
Add your Spotify API credentials to a .env file
Run the application: python app.py

Features

Extract dominant colors using K-means clustering
Map colors to moods based on color psychology
Get song recommendations from Spotify based on detected mood
Simple and intuitive user interface

Note: This app requires server-side processing with Python and access to the Spotify API. GitHub Pages only hosts static content, so you'll need to deploy the full application to a platform like Heroku or Render.
